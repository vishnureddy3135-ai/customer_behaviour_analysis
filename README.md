# customer_behaviour_analysis
data analytics project showcasing customer behaviour analysis using python,sql and power Bi.

## Dataset Overview

The **Customer Shopping Behavior Dataset** contains transactional and demographic information collected from retail customers. The dataset is designed to help analyze customer purchasing patterns, spending habits, product preferences, and overall shopping behavior.

### Dataset Highlights

| Metric                 | Value    |
| ---------------------- | -------- |
| Total Records          | 3,900    |
| Total Features         | 18       |
| Total Sales Revenue    | $233,081 |
| Average Purchase Value | $59.76   |
| Missing Values         | 0        |

### Data Categories

#### Customer Information

* Customer ID
* Age
* Gender
* Location

#### Product Information

* Item Purchased
* Category
* Size
* Color
* Season

#### Transaction Information

* Purchase Amount (USD)
* Payment Method
* Shipping Type

#### Customer Behavior

* Previous Purchases
* Frequency of Purchases
* Subscription Status
* Discount Applied
* Promo Code Used

#### Customer Feedback

* Review Rating

### Project Goal

The primary objective of this analysis is to uncover valuable business insights related to customer behavior, purchasing trends, product performance, and revenue generation. The findings can support data-driven decision-making for marketing, inventory management, and customer retention strategies.

### Expected Outcomes

* Identify top-performing product categories
* Analyze customer spending patterns
* Evaluate seasonal sales trends
* Measure the impact of discounts and promotions
* Understand customer loyalty and repeat purchases
* Generate interactive dashboards and business reports

*resource : kaggle (data set)
